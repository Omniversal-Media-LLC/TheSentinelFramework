# Technology Stack

## Programming Languages
- **Python**: Core AI federation modules, safety layers, and automation scripts
- **JavaScript**: Interactive web components, animations, and evidence filtering
- **HTML/CSS**: Static site generation and responsive design
- **TypeScript**: Type-safe components in Astro framework
- **Shell Scripts**: Deployment automation and system integration

## Web Frameworks
- **Static HTML/CSS/JS**: Primary Sentinel Framework website
- **Astro + React Islands**: EverLightOS public documentation site
- **Tailwind CSS**: Utility-first styling for modern interfaces

## AI Integration
- **Amazon Q**: Primary federation partner for compliance monitoring
- **Cedric AI**: Risk assessment and corrective recommendations
- **QuickSight**: Data-driven analysis and visualization
- **Multi-platform validation**: Cross-system consistency verification

## Data Management
- **CSV**: Master chronology, evidence indices, and email databases
- **JSON**: Configuration files and structured data
- **PDF**: Legal documents, certificates, and formal submissions
- **Markdown**: Documentation, manifests, and content management

## Deployment Infrastructure
- **Static Hosting**: Compatible with Cloudflare Pages, GitHub Pages
- **CDN Optimization**: Global content delivery and performance
- **Local Development**: Python HTTP server for testing

## Development Commands
```bash
# Local development server
cd sentinel-framework
python -m http.server 8000

# Astro development (EverLightOS)
cd everlightos-astro
npm install
npm run dev

# Evidence processing
python FAILSAFE.py
```

## Dependencies
- **Python 3.x**: Core runtime for AI modules
- **Node.js**: Astro framework and build tools
- **Modern browsers**: ES6+ support for interactive features
- **Static hosting**: No server-side requirements
