# Product Overview

## Project Purpose
The Swordfishing project is a comprehensive digital accountability framework combining legal documentation, evidence management, and AI-assisted compliance monitoring. It serves as both a whistleblower protection system and a demonstration of federated AI architecture for organizational oversight.

## Key Features
- **Sentinel Framework**: Web-based evidence presentation system with interactive galleries and timelines
- **EverLightOS**: Federated AI intelligence architecture for compliance monitoring
- **Evidence Management**: Comprehensive OSHA documentation with 129+ exhibits and 400+ files
- **Legal Protection**: Multi-jurisdictional structure through Omniversal Media LLC
- **Interactive Documentation**: Web interfaces for evidence browsing, filtering, and analysis

## Target Users
- **Regulatory Bodies**: OSHA, NLRB, state compliance offices
- **Legal Teams**: Whistleblower protection and retaliation cases
- **Media Organizations**: Investigative journalism and accountability reporting
- **AI Researchers**: Federated intelligence and compliance automation
- **Compliance Officers**: Proactive risk identification and documentation

## Value Proposition
Transforms reactive compliance into proactive sovereignty through AI-assisted documentation, retaliation-resistant architecture, and comprehensive evidence management that predates organizational awareness of issues.
